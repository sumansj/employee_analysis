Aim:
This is an Employee Analysis project.

Technology Used:
---------------------------------
SQL Server Management Studio
MS SQL Server
SQL Server Integration Services(SSIS)

Employee Management Systems:(Project Synopsis)
------------------------------------------------------------------------
1. Create Tables(Region,Country,Job,Department,Employee) to store data.
2. Create Indexex to improve the query performance.
3. Create Views to based on Employee data.
4. Analysing the data using Sub query, Group by clause along with aggregate functions.
5. Implementing different types of Joins to query data by combining tables.
6. Build Stored Procedures to update the data to different tables.
7. Build and Implement SSIS packages.
